Aplicación de Realidad Aumentada

Crear un prototipo de una aplicación de realidad aumentada con alguna de las siguientes características:
- Detección de imágenes
- Detección de planos
- Detección de Rostros
- Spawneo y manipulación de objetos 3D
- UI interactiva ( menús, controles de objetos )

Entrega:
- Repositorio en GIT
- Video en YouTube
- Instalable en Android o IOS
